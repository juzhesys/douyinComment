# 抖音采集机
采集抖音指定账号的所有短视频作品，所有短视频的评论数据等。

## 介绍

2024-03-20   支持所有douyin短视频，评论，粉丝，关注，点赞，转发，收藏等等数据，一键导致自定义分析。


## 功能 
- 评论采集
  - 支持“关键词”搜索作品关键字，得到包含关键词的作品和作者信息
  - 添加/导入需要采集的账号列表，以 “MS4……”开头
  - 添加/导入需要采集的视频列表，视频ID为19位纯数字
  - 按需采集账号下指定数量的视频信息
  - 按需采集视频下指定数量的评论信息
  - 按需采集评论信息，如指定省份地区、指定包含关键字的评论等
  - 支持导出数据作二次统计分析
  - 支持登录账号采集评论信息（未登录只能采集前20条）
- 关注/粉丝采集**（账号设置隐藏列表无法采集）**
  - 添加/导入需要采集的账号列表，以 “MS4……”开头
  - 支持采集条件设置，如采集数量，去重，包含手机号等
  - 支持登录采集粉丝/关注列表
